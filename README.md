## Русский

### О проекте

**Yume no Sakura** («Сон о сакуре») — портфолио-сайт посвящённый красоте Японии: её пейзажам, флоре и духовным местам. Проект создан с нуля на чистом HTML, CSS и JavaScript — без фреймворков и библиотек.

### Что внутри

- Анимированные лепестки сакуры на Canvas
- Плавные scroll-reveal анимации при прокрутке
- 3D tilt-эффект на карточках при наведении
- Эффект печатающегося текста на hover
- Bento-сетка в разделе «О Японии»
- Карточки мест с раскрывающимися описаниями
- Сезонная лента с таймлайном
- Адаптивная вёрстка под мобильные устройства
- Glassmorphism и backdrop-filter эффекты

### Структура

```
Yume-no-Sakura/
├── index.html       — разметка
├── japan.css        — стили
├── japan.js         — анимации и логика
└── sakuraHERO.png   — фоновое изображение героя
```

### Запуск

Просто открой `index.html` в браузере — никаких зависимостей и сборки не требуется.

Или через Live Server в VS Code:
```
1. Установи расширение Live Server
2. Правой кнопкой на index.html → Open with Live Server
```

## English

### About

**Yume no Sakura** ("Dream of Cherry Blossoms") is a portfolio website dedicated to the beauty of Japan — its landscapes, flora, and spiritual places. Built from scratch using pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

### Features

- Animated sakura petals falling on Canvas
- Smooth scroll-reveal animations on scroll
- 3D tilt effect on cards on hover
- Typewriter text effect on hover
- Bento grid layout in the About section
- Place cards with expanding descriptions
- Seasonal timeline
- Fully responsive for mobile devices
- Glassmorphism and backdrop-filter effects

### Structure

```
Yume-no-Sakura/
├── index.html       — markup
├── japan.css        — styles
├── japan.js         — animations and logic
└── sakuraHERO.png   — hero background image
```

### Running

Just open `index.html` in any browser — no dependencies or build step required.

Or via Live Server in VS Code:
```
1. Install the Live Server extension
2. Right-click index.html → Open with Live Server
```

### Tech Stack

| | Technology |
|---|---|

| Styles | CSS3 (Custom Properties, Grid, Flexbox, Backdrop-filter) |
| Logic | Vanilla JavaScript (Canvas API, IntersectionObserver) |
| Fonts | Google Fonts — Cormorant Garamond, Noto Serif JP, Noto Sans JP |
